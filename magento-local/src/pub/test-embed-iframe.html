Dealer Locator Map - Iframe Embed Examples

Example 1: Full Width with Fixed Height (600px)

This is the standard embed with 100% width and 600px height. It adapts to the container width.

<iframe src="dealer-map-embed.html" 
        width="100%" 
        height="600" 
        frameborder="0" 
        style="border: 0; width: 100%; height: 600px;"
        allowfullscreen>
</iframe>

Example 2: Fixed Width and Height (800px x 500px)

This example uses fixed dimensions. Good for sidebars or specific layout requirements.

<iframe src="dealer-map-embed.html" 
        width="800" 
        height="500" 
        frameborder="0" 
        style="border: 1px solid #ddd; width: 800px; height: 500px;"
        allowfullscreen>
</iframe>

Example 3: Responsive 16:9 Aspect Ratio

This responsive embed maintains a 16:9 aspect ratio and scales with the container width. Perfect for responsive designs.

<div style="position: relative; width: 100%; padding-bottom: 56.25%; height: 0; overflow: hidden;">
    <iframe src="dealer-map-embed.html" 
            style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: 0;"
            frameborder="0" 
            allowfullscreen>
    </iframe>
</div>

Example 4: Small Embed (400px x 300px)

A compact version for smaller spaces or widget areas.

<iframe src="dealer-map-embed.html" 
        width="400" 
        height="300" 
        frameborder="0" 
        style="border: 2px solid #333; border-radius: 8px; width: 400px; height: 300px;"
        allowfullscreen>
</iframe>

Example 5: Full Viewport Height

This example uses viewport height (80vh) to create a large, immersive map experience.

<iframe src="dealer-map-embed.html" 
        width="100%" 
        height="800" 
        frameborder="0" 
        style="border: 0; width: 100%; height: 80vh; min-height: 500px;"
        allowfullscreen>
</iframe>

Embed Features

The dealer locator map embed includes:

  • Interactive Map: Fully functional Google Maps with custom styling
  • Search Functionality: Location search with autocomplete
  • Tag Filtering: Filter dealers by tags/categories
  • Dealer Information: Complete dealer details including address, phone, email, and website
  • Responsive Design: Works on all screen sizes
  • Clustering: Groups nearby markers for better performance
  • Results Panel: Shows up to 3 dealers per row in a centered grid
  • Get Directions: Direct links to Google Maps for navigation

Integration Tips

Responsive Design

For responsive websites, use percentage widths or the responsive wrapper technique shown in Example 3.

Custom Styling

You can add custom CSS to style the iframe container:

.map-container {
    border: 2px solid #ddd;
    border-radius: 8px;
    overflow: hidden;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

Performance

Consider lazy loading for better page performance:

<iframe src="dealer-map-embed.html" 
        loading="lazy"
        width="100%" 
        height="600">
</iframe>
